NeuroMatrix3D

NeuroMatrix3D converts plain text into downloadable 3D-printable braille STL files for tactile reading materials, braille labels, classroom resources, signs, and accessibility projects.

Source code: GitHub Repository

Advanced Parameters

Instructions

  1. Enter text into the text to the braille STL generator.
  2. Adjust dot radius, spacing, page thickness, and page width if needed.
  3. Generate a 3D-printable braille STL file.
  4. Download the generated file and prepare it in slicing software for 3D printing of accessible braille materials.

Designed For

NeuroMatrix3D targets educators, makers, accessibility researchers, libraries, parents, and students. It enables creation of tactile reading materials, braille labels for 3D printing, and other accessible 3D-printed braille resources, while supporting UEB Grade 2 braille STL generation.
